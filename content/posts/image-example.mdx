# 画像の使い方サンプル

このブログでは、MDX内で画像を簡単に使用できます。

## ローカル画像の使用方法

ローカル画像は `public/images/posts/` フォルダに配置して、以下のように記述します：

```markdown
![説明文](ファイル名.jpg)
```

例：
```markdown
![美しい風景](landscape.jpg)
```

## 外部画像の使用方法

外部URLの画像も直接使用できます：

![Unsplashのサンプル画像](https://images.unsplash.com/photo-1506905925346-21bda4d32df4?w=800&q=80)

## 画像のサイズと配置

画像は自動的にレスポンシブ対応され、最適なサイズで表示されます。

### 使い方のポイント

1. **ローカル画像**: `public/images/posts/` にファイルを配置
2. **外部画像**: HTTPSのURLを直接指定
3. **alt テキスト**: SEOとアクセシビリティのために必ず記述
4. **画像形式**: JPG, PNG, WebP, AVIF に対応

## コード例

```markdown
<!-- ローカル画像 -->
![記事のサムネイル](thumbnail.png)

<!-- 外部画像 -->
![外部画像の例](https://via.placeholder.com/800x400)
```

## 注意事項

- 画像は自動的に最適化されます
- 遅延読み込み（Lazy Loading）が自動的に適用されます
- WebPやAVIF形式に自動変換されます（ブラウザが対応している場合）

これで、ブログ記事に画像を自由に追加できます！
