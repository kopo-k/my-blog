## Next.js App Routerとは

Next.js 13で導入されたApp Routerは、Reactの最新機能を活用した新しいルーティングシステムです。Server ComponentsやStreaming、Suspenseなどの機能を簡単に使えるようになりました。

## 主な特徴

### 1. Server Components

デフォルトでServer Componentsとして動作し、必要な部分だけClient Componentにすることで、バンドルサイズを削減できます。

```tsx
// Server Component (デフォルト)
export default async function Page() {
  const data = await fetch('https://api.example.com/data')
  return <div>{data}</div>
}
```

### 2. ネストされたレイアウト

レイアウトファイルを使って、ページ間で共通のUIを簡単に共有できます。

```tsx
// app/layout.tsx
export default function RootLayout({
  children,
}: {
  children: React.ReactNode
}) {
  return (
    <html lang="ja">
      <body>{children}</body>
    </html>
  )
}
```

### 3. データフェッチング

`async`/`await`を使った直感的なデータフェッチングが可能です。

```tsx
async function getData() {
  const res = await fetch('https://api.example.com/data', {
    cache: 'no-store' // or 'force-cache'
  })

  if (!res.ok) {
    throw new Error('Failed to fetch data')
  }

  return res.json()
}
```

## まとめ

App Routerは、モダンなReactアプリケーション開発のベストプラクティスを簡単に実装できる強力な機能です。Server Componentsを活用することで、パフォーマンスとユーザー体験の向上が期待できます。
