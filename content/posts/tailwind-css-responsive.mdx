## Tailwind CSSとは

Tailwind CSSは、ユーティリティファーストのCSSフレームワークです。事前に定義されたクラスを組み合わせることで、カスタムCSSを書かずにデザインを実装できます。

## レスポンシブデザインの基本

### ブレークポイント

Tailwind CSSには以下のデフォルトブレークポイントがあります：

- `sm`: 640px以上
- `md`: 768px以上
- `lg`: 1024px以上
- `xl`: 1280px以上
- `2xl`: 1536px以上

### 基本的な使い方

```html
<div class="text-sm md:text-base lg:text-lg">
  画面サイズに応じてテキストサイズが変わります
</div>
```

## 実践的な例

### レスポンシブグリッド

```jsx
<div className="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">
  <div className="bg-white p-6 rounded-lg shadow">
    <h3 className="text-lg font-semibold">カード1</h3>
    <p className="text-gray-600">コンテンツ</p>
  </div>
  <div className="bg-white p-6 rounded-lg shadow">
    <h3 className="text-lg font-semibold">カード2</h3>
    <p className="text-gray-600">コンテンツ</p>
  </div>
  <div className="bg-white p-6 rounded-lg shadow">
    <h3 className="text-lg font-semibold">カード3</h3>
    <p className="text-gray-600">コンテンツ</p>
  </div>
</div>
```

### フレックスボックスレイアウト

```jsx
<div className="flex flex-col md:flex-row gap-4">
  <div className="flex-1">
    <img src="/image.jpg" alt="画像" className="w-full h-auto" />
  </div>
  <div className="flex-1 p-4">
    <h2 className="text-2xl font-bold mb-4">タイトル</h2>
    <p className="text-gray-700">説明文...</p>
  </div>
</div>
```

## モバイルファーストアプローチ

Tailwind CSSはモバイルファーストで設計されています。基本スタイルをモバイル向けに書き、大きな画面向けに上書きしていきます。

```jsx
<button className="
  px-4 py-2
  text-sm
  md:px-6 md:py-3
  md:text-base
  lg:px-8 lg:py-4
  lg:text-lg
  bg-blue-500 text-white rounded
">
  ボタン
</button>
```

## ユーティリティクラスの活用

### スペーシング

```jsx
<div className="p-2 md:p-4 lg:p-6">
  <div className="mb-2 md:mb-4">
    <h1 className="text-xl md:text-2xl lg:text-3xl">見出し</h1>
  </div>
</div>
```

### 表示・非表示の制御

```jsx
<nav className="hidden md:block">
  {/* デスクトップメニュー */}
</nav>

<button className="md:hidden">
  {/* モバイルメニューボタン */}
</button>
```

## まとめ

Tailwind CSSのレスポンシブ機能を活用することで、効率的にモバイル対応のWebサイトを構築できます。ユーティリティクラスを組み合わせることで、複雑なレイアウトも簡単に実装可能です。
