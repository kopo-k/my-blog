# VSCode生産性向上テクニック

Visual Studio Codeを使いこなして開発効率を上げましょう。

## 必須ショートカット

### ファイル操作
- `Cmd/Ctrl + P`: クイックオープン
- `Cmd/Ctrl + Shift + P`: コマンドパレット
- `Cmd/Ctrl + B`: サイドバーの表示/非表示

### 編集操作
- `Alt + ↑/↓`: 行の移動
- `Shift + Alt + ↑/↓`: 行の複製
- `Cmd/Ctrl + D`: 次の一致を選択
- `Cmd/Ctrl + Shift + L`: すべての一致を選択

## おすすめ拡張機能

### 開発効率化
1. **Prettier**: コード整形
2. **ESLint**: JavaScript/TypeScriptのリンター
3. **GitLens**: Git履歴の可視化
4. **Live Server**: ローカルサーバーの起動

### テーマ・見た目
1. **Tokyo Night**: 人気のダークテーマ
2. **Material Icon Theme**: ファイルアイコン

## 設定のカスタマイズ

```json
{
  "editor.fontSize": 14,
  "editor.tabSize": 2,
  "editor.formatOnSave": true,
  "editor.minimap.enabled": false,
  "workbench.colorTheme": "Tokyo Night"
}
```
